Hipertextos + Imagen

Primer caso de prueba del servidor TWS. La página combina marcado HTML con un recurso binario (PNG) servido en la misma sesión. Verifica el parsing de la línea de solicitud GET, la detección de tipo MIME y el log de cada petición.

Método
GET
Estado
200 OK
Content-Type
text/html
Recursos servidos
2 · HTML + PNG

¿Qué demuestra este caso?

El servidor recibe una petición GET /case1/index.html, la decodifica a través de la máquina de estados HTTP y entrega el archivo solicitado. A su vez, el navegador dispara una segunda petición para la imagen embebida.

  • Parsing correcto de la línea GET /ruta HTTP/1.1 y de las cabeceras.
  • Detección de MIME por extensión: .html → text/html, .png → image/png.
  • Cálculo de Content-Length usando stat() antes del envío.
  • Registro en log de IP, método, ruta, código de estado y bytes transferidos.
  • Cada conexión se atiende en un hilo dedicado vía pthread_create().
